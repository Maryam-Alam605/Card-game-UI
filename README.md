# 🐾 Cat Lovers Memory Game

A fun and adorable card-matching memory game for cat lovers!  
Flip the paw cards to find matching cats hidden behind them.

## 🎯 Features
- 8 unique cat pairs
- Simple, clean UI
- Built with HTML, CSS, and JavaScript
